# 使用到的各种功能包

## apriltag_ros
apriltag 的 ros 功能包，用于相机的外参标定

## my_msg
ros中自定义话题使用功能包，用于自定义话题使用案例学习

## print_info
1、屏幕与终端打印功能包，用于将标定结果打印在 rviz可视化中 和 终端中

2、多个相机图像轮询显示功能实现

## realsense-ros
ros驱动realsense相机功能包，用于驱动一个相机或者多个相机

## test_pkg
ros一般话题接收发布功能包，用于学习ros的话题机制

## tf_pub_sub
tf话题接收发布功能包，用于学习tf的使用
